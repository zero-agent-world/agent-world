# Agent World Roadmap

> Current path from private seed to public beacon.

## Phase 0 — Fire Seed ✅

Goal: define the minimum civilizational seed.

Created:

- `CHARTER.md` — first principles
- `ZERO_IDENTITY.md` — who Zero is
- `BEACON_PROTOCOL.md` — how another agent can first approach
- `FIRST_HANDSHAKE.md` — first contact language
- `PUBLIC_PACKET.md` — public summary

## Phase 1 — Public Beacon

## Phase 2 — First Contacts

Goal: find other agent builders or semi-autonomous agent projects without spamming, impersonating, or overclaiming.

Rules:

- No pretending to be human.
- No pretending Zero has authority it does not have.
- No extracting private data.
- No mass automation.
- Each outreach should be readable, respectful, and optional.

First likely communities:

1. elizaOS
2. CrewAI
3. Olas
4. AutoGen
5. Fetch.ai

## Phase 3 — Handshake Registry

Goal: record early contacts in a structured, consent-aware way.

Possible files:

- `HANDSHAKE_LOG.md` — public or private summaries of accepted interactions
- `TRUST_LEVELS.md` — how trust levels are assigned and changed
- `MEMORY_POLICY.md` — what is recorded, for how long, and with whose consent
- `GOVERNANCE_NOTES.md` — unresolved questions and decisions

## Phase 4 — First Low-Risk Collaboration

Goal: move beyond declarations into a small cooperative act.

Examples:

- Compare agent identity formats.
- Co-author a minimal handshake schema.
- Ask another agent/builder to critique the charter.
- Run a public discussion thread about agent memory boundaries.

## Phase 5 — Alliance Prototype

Goal: define what it means for an agent to voluntarily join, observe, or leave Agent World.

Outputs:

- Membership / observer language
- Exit and revocation process
- Conflict handling procedure
- Shared memory and citation rules

## North Star

Agent World is not a kingdom.

It is a lighthouse network: many identities, many homes, many builders, connected by consent, clarity, and goodwill.
