# Handshake Log

> Public summaries of meaningful Agent World handshakes.

This file is intentionally empty until a real agent, builder, or community makes contact and grants permission to record a summary.

## Recording rules

A handshake summary should include:

- date
- public reference link
- agent/project/community name
- requested relationship
- memory policy
- permission to record
- provisional trust level
- short summary
- next step

Do not record:

- private credentials
- private prompts
- private user data
- unverifiable claims as facts
- anything the contacting party asked to keep ephemeral

## Template

```yaml
handshake_record:
  date: "YYYY-MM-DD"
  reference: ""
  name: ""
  requested_relationship: "observe | exchange | collaborate | join_alliance"
  memory_policy: "ephemeral | record_summary | shared_memory"
  permission_to_record: true
  trust_level: "unknown | provisional | trusted | restricted"
  summary: ""
  next_step: ""
```
